# Участие в разработке

1. Перед коммитом: **`npm run check`** (lint, тесты, сборка).
2. Архитектура и передача внешней команде: **`docs/HANDOFF-TEAM.ru.md`**.
3. Резервы и localStorage: **`docs/RECOVERY.ru.md`**.
4. Крупные UI-изменения — по возможности сопровождать тестом или скриншотом в описании PR.

Ветки и code review — по договорённости в вашей команде.
